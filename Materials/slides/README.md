# 🎨 Chapter 9: Spectacular Wonders - Interactive Slideshow

## 📊 Slideshow Guide

This folder contains interactive HTML slideshows for the class presentations. Follow the instructions below to view and use them.

---

## 🚀 How to View the Slideshows

### Method 1: Click and Open (Easiest)
1. Click on any `.html` file in this folder
2. GitHub will display a preview
3. Click "View Raw" to open in full screen
4. Or download and open on your computer

### Method 2: Download to Your Computer
1. Right-click on the `.html` file
2. Select "Save Link As..."
3. Open the downloaded file with your web browser
4. Works without internet connection

### Method 3: Share Link
1. Click on the `.html` file
2. Click "Raw" button
3. Copy the URL
4. Share with classmates

## 🎯 Discussion Questions

After each slideshow, think about:

**After Slideshow 1:**
- What makes something a "wonder"?
- Can you think of other wonders not mentioned?

**After Slideshow 2:**
- Why do you think only the pyramid still stands?
- Which ancient wonder interests you most?

**After Slideshow 3:**
- How are natural wonders different from man-made ones?
- Have you seen any of these wonders?

**After Slideshow 4:**
- What makes descriptive writing powerful?
- How would you describe your favorite place?
